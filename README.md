# JavaScript Fundamentals Practice

![JavaScript](https://img.shields.io/badge/JavaScript-ES6+-yellow.svg)
![License](https://img.shields.io/badge/License-MIT-blue.svg)

A collection of JavaScript exercises covering variable declarations, data types, loops, arrays, and objects.

## Features

- Variable declarations with `var`, `let`, and `const`
- Type conversion and dynamic boolean assignment
- Object creation with nested properties
- Array manipulation including:
  - Manual string reversal
  - Transformation to uppercase
  - Filtering and counting elements
- Proper error handling demonstration

## Tasks Completed

### Basic Exercises (10 pts each)
1. `var` name declaration
2. `let` age declaration (as string)
3. `const` demonstration (with reassignment error)
4. Dynamic `isStudent` boolean assignment
5. Object with nested properties
6. Array with color strings (including duplicates)

### Challenges (20 pts each)
1. **Reverse and Transform**: Manual string reversal + uppercase conversion
2. **Filtering and Counting**: Even numbers greater than 20
